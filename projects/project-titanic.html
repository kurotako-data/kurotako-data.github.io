Project Titanic

Predicting Passenger Survival Based on Nationality, Class, and Gender

Introduction

This project aims to explore the survival rates of Titanic passengers by analyzing key factors such as nationality, class, and gender. The primary objective is to build a predictive model that combines these variables to forecast the likelihood of survival.

What datasets are used in this project?

For this analysis, we use the well-known Titanic dataset. This dataset includes information on the passengers aboard the Titanic, which allows for an in-depth analysis of the factors that influenced survival. Key variables include:

  • Passenger Information: Includes details such as name, gender, and nationality (derived from passenger names or other methods).
  • Class: The passenger’s class of travel (1st, 2nd, or 3rd class).
  • Survival Status: A binary variable indicating whether the passenger survived (1) or not (0).
  • Other Variables: Age, fare, and embarkation point.

Data Volumetrics

  • Passengers: 891 rows / 12 columns

Project Objectives

The primary objectives of this project are:

  • Predict Survival Based on Nationality: Analyze the impact of a passenger’s nationality on their likelihood of survival.
  • Explore the Role of Class and Gender: Investigate how class and gender further influence survival rates.
  • Combine Factors: Develop a global predictive model that integrates nationality, class, and gender to forecast survival.

Next Steps

The next stages of this project include:

  • Data cleaning and preparation (including dealing with missing values).
  • Exploratory data analysis (EDA) to uncover key patterns and relationships.
  • Building classification models to predict survival based on nationality, class, and gender.
  • Model evaluation and optimization for better accuracy and generalization.
